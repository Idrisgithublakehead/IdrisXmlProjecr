# IdrisXmlProjecr
Connect 3 Xml Game
Android Connect 3 Game

An Android game application built using XML-based layouts and Kotlin/Java in Android Studio.

Overview

This project implements a Connect 3 game featuring structured UI design and interactive game logic. The application demonstrates understanding of Android activity lifecycle, user input handling, and XML layout structuring.

Features

Interactive Connect 3 gameplay

XML-based UI using ConstraintLayout

Dynamic user interaction handling

Structured project architecture

AndroidManifest configuration

Technologies Used

Kotlin / Java

XML

Android Studio

Gradle
